戦闘と魔法

基本戦闘システム

アリーナの戦闘はリアルタイム方式。マウスクリックで攻撃する。 命中・ダメージはキャラクターのステータスと装備、敵のARによって決定される。

命中判定

攻撃が当たるかどうかは以下の要素で決まる：

要素効果
AGI（敏捷性）攻撃の命中ボーナス / 防御ボーナス（両方向）
LUC（幸運）全行動に影響するランダム修正
レッドガード特性レベル÷3 の命中・ダメージボーナス（近接のみ）
ウッドエルフ特性レベル÷3 の命中・ダメージボーナス（弓のみ）
ダークエルフ特性レベル÷4 の命中・ダメージボーナス（近接・弓）

ダメージ計算

ダメージは武器の基本値 + STRによる修正 + 種族ボーナスで決まる。 敵のアーマーレーティング（AR）が低いほど、ダメージを軽減される。

クリティカルヒット

クリティカルヒット = 通常の3倍ダメージ
盗賊系・一部の魔法使い系クラスがレベルに応じた確率で発動。 暗殺者が最高確率、侵入者・吟遊詩人・ローグなども保有。

セービングスロー（抵抗判定）

呪文・毒・特殊効果を受けたとき、成功すると効果を軽減または無効化できる。 WIL（意志力）が高いほど成功しやすい。

クラスセーブ効果
通常成功でダメージ半減
修道士（Monk）成功でダメージ完全無効
ノルド（Nord）冷気攻撃のセーブ成功でダメージ無効
ブレトン（Breton）魔法攻撃のセーブ成功でダメージ無効

魔法システム

呪文はキャラクターシートの呪文書から選択して使用する。各呪文には呪文ポイント（SP）コストがある。 SPが0になると呪文は使えない。魔法使い系クラスはINT値×倍率のSPを持つ。

呪文ポイント（SP）の倍率

クラスSP = INT ×
魔法使い（Mage）× 2
ソーサラー（Sorcerer）× 3 まで吸収（自己生成なし）
ヒーラー（Healer）× 1.75
戦闘魔術師（Battle Mage）× 1.75
呪文剣士（Spellsword）× 1.5
ナイトブレード（Nightblade）× 1.5
吟遊詩人（Bard）= INT

種別代表的な呪文用途

呪文の作成（Spellmaker）

魔術師ギルドのSpellmakerを使うと、オリジナルの呪文を作成できる。 効果・範囲・持続時間・強度を組み合わせてカスタマイズする。 ヒーラーは防御系呪文が安く、戦闘魔術師は攻撃系呪文が安く作れる。

経験値とレベルアップ

敵を倒したり、クエストを完了することで経験値を獲得する。 レベルアップ時はステータスが増加し、体力も上昇する。魔法使い系はSPも増加。

レベルアップ速度（速い順）：盗賊 → 戦士 → 戦士サブクラス → 盗賊サブクラス → 魔法使い → 魔法使いサブクラス
