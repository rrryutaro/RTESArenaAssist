クラス解説（全18クラス）

シーフ系クラス — 主要ステータス: 敏捷 (AGI) ・ 知性 (INT)

全シーフ系クラスはピッキング・スリ・盗みができる。速度 (SPD) も重要。

シーフ（盗賊）

素早く狡猾。戦闘では奇襲・偵察・クリティカルヒットを狙う。経験値の上昇が最も速い。

★ レベル毎にクリティカルヒット（3倍ダメージ）の確率が増加
武器短剣・ショートソード・ブロードソード・サーベル・ウォーアックス・ショートボウ
防具レザーのみ
盾バックラーのみ
初期体力25 + d10

バーグラー（侵入者）

錠前解錠とエリア侵入の専門家。盾は使えない。

★ クリティカルヒット確率（シーフより低い）
武器短剣・ショートソード・タント・ショートボウ
防具レザーのみ
盾なし
初期体力25 + d8

アサシン（暗殺者）

暗殺特化。弱点を突く能力が高く、自分より強い敵を倒すことも。武器の選択肢は広いがレザー・盾不可。

★ 全クラス中最高のクリティカルヒット確率
武器全武器
防具レザーのみ
盾なし
初期体力25 + d12

ローグ（無法者）

武器訓練も積んだシーフ。戦闘力はウォーリアークラス並みで、ピッキングも可能。

★ 低確率でクリティカルヒット
武器全武器
防具レザー・チェインのみ
盾タワーシールド以外
初期体力25 + d14

アクロバット（軽業師）

壁登り・跳躍・転回が超人的。全クラス中最速の移動。敵の命中率を下げるパッシブ能力あり。

★ 敵の命中基礎確率を一定割合減少 / クリティカルヒット確率あり
武器短剣・ショートソード・ブロードソード・タント・ショートボウ
防具レザーのみ
盾なし
初期体力25 + d8

バード（吟遊詩人）

「なんでも屋」。クリティカル・武器・盗み・魔法すべてこなせる万能クラス。呪文ポイント = INT値。

武器短剣・ショートソード・ブロードソード・サーベル・メイス・ウォーアックス・ショートボウ
防具レザー・チェインのみ
盾タワーシールド以外
初期体力25 + d10
ウォーリアー系クラス — 主要ステータス: 筋力 (STR) ・ 持久力 (END)

基本的に呪文は使えない。敏捷 (AGI) も重要（アーマーレーティングに影響）。

ウォーリアー（戦士）

タムリエルの基盤。全武器・防具・盾を使用可能。経験値上昇はシーフの次に速い。プレートアーマーとその魔法付与版を使える唯一のクラス（ナイト・レンジャーと共に）。

武器全武器
防具全防具
盾全盾
初期体力25 + d20

ナイト（騎士）

貴族の戦士。礼節と名誉を重んじる。武器と防具の修理が自動的に行われる（インベントリ内すべて）。

★ 麻痺（呪文・毒問わず）への完全免疫 / 武器・防具を自動修理
武器全武器
防具レザー以外
盾全盾
初期体力25 + d18

レンジャー（放浪戦士）

追跡・生存・道案内の達人。都市間移動時間が短縮される。攻撃にレベル分のボーナスダメージ。

★ 都市間移動時間を自動短縮 / 攻撃にレベル値のボーナスダメージ
武器全武器
防具全防具
盾タワーシールド以外
初期体力25 + d18

アーチャー（弓使い）

遠距離武器のエキスパート。盾不可。弓でのクリティカルヒット確率あり。

★ 弓使用時にレベル毎のクリティカルヒット確率
武器全武器
防具レザー・チェインのみ
盾なし
初期体力25 + d16

モンク（修道士）

精神と肉体の鍛錬で体を武器化。防具・盾不可。レベルが上がるほど敵の命中率が下がる。呪文セービング成功時は無ダメージ。

★ 敵の命中基礎確率をレベル分減少 / 呪文セーブ成功時ノーダメージ / クリティカルヒット（近接のみ）
武器全武器
防具なし
盾なし
初期体力25 + d14

バーバリアン（野蛮人）

過酷な環境で生き延びてきた戦士。全クラス中最多の初期体力。体力回復量がENDに応じてボーナス。

★ 毒への完全免疫 / ENDに応じた追加体力回復
武器全武器
防具チェインまで
盾全盾
初期体力25 + d30（全クラス最大）
メイジ系クラス — 主要ステータス: 知性 (INT) ・ 意志力 (WIL)

メイジ（魔法使い）

魔法の扱いに特化。呪文ポイント = INT×2。防具・盾不可。攻防ともに呪文が頼り。

武器短剣・杖
防具なし
盾バックラーのみ
初期体力25 + d6

ソーサラー（妖術師）

魔法の「電池」。内部で呪文ポイントを生成しないが、自分に向けられた呪文を吸収してポイントにする。最大吸収量 = INT×3。フル充電時は最強の魔法使い。

★ 自身に向けられた呪文を吸収（ダメージなし、ポイント増加）/ 上限超過時は通常ダメージ / 自己呪文の吸収・自動再生不可
武器全武器
防具チェインまで
盾なし
初期体力25 + d6

ヒーラー（治癒師）

回復特化の魔法使い。防御的な呪文が得意で、呪文制作時の防御呪文コストが安い。呪文ポイント = INT×1.75。自然回復速度ボーナス。

武器短剣・杖・メイス・フレイル
防具レザー・チェインのみ
盾ラウンドシールドまで
初期体力25 + d8

バトルメイジ（戦闘魔術師）

戦闘用攻撃魔法の達人。呪文制作時の攻撃呪文コストが安い。呪文ポイント = INT×1.75。

武器全武器
防具レザーのみ
盾ラウンドシールドまで
初期体力25 + d10

スペルソード（呪文剣士）

防具を装備したまま呪文を使える戦士魔法使い。汎用性が高い。呪文ポイント = INT×1.5。

武器全武器
防具レザー・チェインのみ
盾タワーシールド以外
初期体力25 + d12

ナイトブレード（夜刃使い）

魔法を潜入・諜報・隠密活動に使う夜の使者。ローグ並みのピッキング能力。呪文ポイント = INT×1.5。

★ レベル毎にクリティカルヒット確率
武器短剣・杖・ショートボウ・ショートソード・サーベル
防具レザーのみ
盾バックラーのみ
初期体力25 + d8
